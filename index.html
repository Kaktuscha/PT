10 Essential unix commands

What is Unix
UNIX is an operating system.
By operating system, we mean the suite of programs which make the computer work. It is a stable, multi-user, multi-tasking system for servers, desktops and laptops and etc.
The UNIX operating system is made up of three parts; the kernel, the shell and the programs.
The kernel of UNIX is the hub of the operating system: it allocates time and memory to programs and handles the filestore and communications in response to system calls.
The shell acts as an interface between the user and the kernel(CLI).
Terminal is a program that run a shell.

CP

CP is a Unix shell command to copy files and directories

cp syntax

cp options source destination
option description
cp -a archive files
cp -f force copy by removing the destination file if needed
cp -i interactive - ask before overwrite
cp -n no file overwrite
cp -u update - copy when source is newer than dest

$ cp main.c def.h /home/usr/rapid/

Copy 2 files main.c and def.h to destination absolute path directory /home/usr/rapid/.

$ cp -u * /bak

Update all files in current directory - copy only newer files to destination directory bak.

$ cp src /home/usr/rapid/

Copy directory src to absolute path directory /home/usr/rapid/.

CD

CD is a UNIX command to change the directory/folder of the terminal's shell.

cd syntax

$ cd [directory]

Examples:

$ cd

Change to home directory (determined by $HOME environment variable)

$ cd ..

Change to parent directory.

$ cd /home/user/Desktop

Change to directory with absolute path /home/user/Desktop.

ls

LS is a UNIX shell command that lists directory contents of files and directories.

ls syntax

$ ls [options] [file|dir]

Options

option description
ls -a list all files including hidden file starting with '.'
ls -l list with long format - show permissions
ls -r list in reverse order
ls -s list file size
ls -S sort by file size
ls -t sort by time & date

Examples:

$ ls -l

List with long format

mv

MV command is used to move files and directories.

ls syntax

$ mv [options] source destination

Options

option description
mv -f force move by overwriting destination file without prompt
mv -i interactive prompt before overwrite
mv -u update - move when source is newer than destination
mv -v verbose - print source and destination files

Examples:

$ mv main.c def.h /home/usr/rapid/

Move main.c def.h files to /home/usr/rapid/ directory.

rm

RM remove files or directories.

rm syntax

$ rm [options] file

Options

option description
rm -f ignore nonexistent files, never prompt
rm -i prompt before every removal
mv -v explain what is being done

Examples:

$ rm /home/hope/file.txt

Delete the file "file.txt" that located in the directory "/home/hope".

cat

CAT command is used to display the content of text files and to combine several files to one file

rm syntax

$ rm [options] file1 [file2...]

Options

option description
cat -s squeeze blank lines to one line
cat -E show $ at the end of line
cat -T show ^I instead of tabs

Examples:

$ cat list1.txt

milk
bread
apples

$ cat list2.txt

house
car

View text file data.

$ cat list1.txt list2.txt > list3.txt

milk
bread
apples
house
car

tar

The UNIX “tar” stands for tape archive. The tar is most widely used command to create compressed archive files.

tar syntax

$ tar [options] destination source

Options

A tar files to an archive.
c Create a new archive.
d Calculate any differences between the archive and the file system.
t List the contents of an archive.
x Extract files from an archive.

Examples:

$ tar -cf archive.tar file1 file2

Creates archive.tar from file1 and file2.

grep

The grep filter searches a file for a particular pattern of characters, and displays all lines that contain that pattern. The pattern that is searched in the file is referred to as the regular expression (grep stands for globally search for regular expression and print out).

grep syntax

$ grep [options] pattern [files]

Options

c This prints only a count of the lines that match a pattern
i Ignores, case for matching
n Display the matched lines and their line numbers
v This prints out all the lines that do not matches the pattern

Examples:

$ cat > 1.txt

unix is great os. unix is opensource. unix is free os.
learn operating system.
Unix linux which one you choose.
uNix is easy to learn.unix is a multiuser os.Learn unix .unix is a powerful.

Creates 1.txt file with given text.

$ grep -i "UNix" 1.txt

OUTPUT

unix is great os. unix is opensource. unix is free os.
Unix linux which one you choose.
uNix is easy to learn.unix is a multiuser os.Learn unix .unix is a powerful.

The -i option enables to search for a string case insensitively in the give file. It matches the words like “UNIX”, “Unix”, “unix”.

PWD

PWD stands for Print Working Directory. It prints the path of the working directory, starting from the root.

pwd syntax

$ pwd [OPTION]

Options

L Prints the symbolic path
P Prints the actual path

Examples:

$ pwd

/home/hope/mysymlink/mydir

Print the name of the working directory.

MAN

Man command in Unix is used to display the user manual of any command that we can run on the terminal. It provides a detailed view of the command which includes NAME, DESCRIPTION, OPTIONS, RETURN VALUES, ERRORS, etc.

man syntax

$ man [OPTION] [COMMAND NAME]

Options

f file Display a summary (one line) for each file
k keywords Displays the header lines that contain any of the keywords

Examples:

$ man printf

It displays the whole manual of the command
